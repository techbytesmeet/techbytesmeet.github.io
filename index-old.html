What We Do

12 weeks program

  • Welcome: Getting everyone's familiarity with the topics
  • Projects & Workshops: Completing Coding & Design Workshops and Projects
  • Leadership in Tech: Preparing exit presentation on topic of choice

Goals & Accomplishments

  • Fostering Creativity: We are creating a platform for fostering Creativity, Leadership and Entrepreneurship in Emerging Technologies among youths.
  • Creators: We are creating creators which is evident by the Animation, Graphics, Infographics, 3D print designs, VR designs, AR designs, Games created using Python, Designs created using CodeBlocks, Data Visualization, Data Analysis projects done by our interns (which is posted in our Social Media’s as well)
  • Empower Next Gen Youth: Our goal is to Engage, Educate, Empower the Next Generation of Youths in STEM.

How We Do It

Skills distribution among the 12 weeks of program

  • 30% Coding & Design Thinking: Coding (Python), Data Science, Machine Learning, Artificial Intelligence, Internet of Things, 3D Printing, Virtual Reality, Augmented Reality design
  • 40% Project Building: Creating projects with the skills you’ve learned through fun weekly challenges
  • 20% Leadership: Demonstrating how to work with and lead a technical team
  • 10% Presentation: Presenting your work to a non-technical audience

What You Will Learn

STEM Skills

  • Coding: Visual Block based Coding
  • ML/AI: ML, AI algorithms & apply them for solving problems
  • Data Science: Data Analysis, Data Visualization skills for solving real problems
  • Design Thinking: Infographics & 2D Animation, 3D Printing design, VR/AR, Web/App Design, Digital Marketing

Paths You Can Follow

These STEM skills can unlock paths for various fields

  • Coding Jobs: Automation Engineer, Software Engineer, Web Developer...
  • Design Thinking: Game Developer, UX Designer, Content Creator...
  • ML/AI: ML Engineer, NLP Scientist, Data Engineer...
  • Data Science: Data Scientist, Business Analyst, Data Architect...
